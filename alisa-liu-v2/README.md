# Direction: Alisa Liu (fresh run)

A research direction of the [Self-Driving Research Notes](../README.md) project. Reading notes on NLP
researcher **[Alisa Liu](https://alisawuffles.github.io/)** — built by running our method **completely
from scratch** (starting from only a Google Scholar URL, with no reuse of the earlier
[`alisa-liu/`](../alisa-liu/) direction) to test that the workflow reproduces.

- **[`index.html`](index.html)** — the content: a "what I find most interesting" opening, research-arc
  timeline, six foldable per-paper deep dives (Intro / Method / Results / Future + "what's missing"),
  full publication list, discussion questions.
- **[`methodology.html`](methodology.html)** — the method as applied here (two-pass: 3-agent broad map →
  verify gate → 6-agent per-paper deep read), and what the fresh run independently established/corrected.

Open `index.html` in a browser. No build step, no dependencies.

## Honesty

Built from her homepage + CV, DBLP, the ACL Anthology, and arXiv (Google Scholar wasn't reliably
fetchable, so citation counts are omitted). **SuperBPE** and **Data Mixture Inference** are co-first
("equal contribution"); same-name researchers in HCI/genomics/neuroscience were excluded. A few 2026
items are preprints. Every paper links its primary source — please verify before citing.
