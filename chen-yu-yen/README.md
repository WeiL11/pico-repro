# Direction: Chen-Yu (Robin) Yen

A research direction of the [Self-Driving Research Notes](../README.md) project — the **first built under the
v0.6 method**. Reading notes on **Chen-Yu (Robin) Yen** (NYU ECE PhD, advisor H. Jonathan Chao; RL for
Internet congestion control, with a pivot to ML for MRI; now ByteDance).

- **[`index.html`](index.html)** — the content: a "what I find most interesting" opening, research-arc
  timeline, four foldable per-paper deep dives (Orca, DeepCC, Sage, ASMR) with Intro/Method/Results/Future
  + "what's missing", a full publication list, discussion questions, and a completeness/honesty box.
- **[`methodology.html`](methodology.html)** — how it was built (v0.6: slim Pass 1 → verify → pipelined
  4-agent Pass 2 → completeness critic + quality gate), with telemetry.
- **[`_research/`](_research/)** — the archived deep-read evidence (one file per paper), so every claim on
  the content page is traceable.
- **[`DECISIONS.md`](DECISIONS.md)** — the decision log (paper selection, disambiguation, budgets).

Open `index.html` in a browser. No build step.

## Honesty
Two SIGCOMM papers (Orca, Sage) were paywalled to the research agents, so their exact %-numbers are marked
unverified (method reconstructed from the authors' survey + open-source code + independent benchmarks).
Author orders differ across his co-first papers (Orca/DeepCC: Yen 2nd-listed; Sage: 1st-listed). The 2014
graphics paper is most likely a namesake. Citation counts are approximate (Google Scholar).
